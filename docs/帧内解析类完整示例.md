# 帧内解析类（ThinkGear / 思知金牛座 Payload / TLV）— 完整示例代码

> 本文**仅文档**：可复制到工程中的 `thinkgear/` 或 `protocol/` 目录，并在 **CMake** 中加入对应 `.h/.cpp`。  
> **前置**：`ThinkGearFrameAssembler` 已发出 **完整帧** `[0xAA][0xAA][N][Payload×N][Checksum]`（组帧内已按 Payload 求和取反验证校验和）。本类只解析 **Payload** 内的代码字与数据。  
> **协议对照**：思知教程中的**小包** `AA AA 04 80 02 xxHigh xxLow chk`、**大包** `AA AA 20 …`（32 字节 Payload）见 `docs/脑电串口解析与缓冲设计教学.md` 第〇节。

---

## 一、职责与数据流

```
frameReady(QByteArray 整帧)
        ↓
ThinkGearPayloadParser::parseFrame
        ↓ 校验帧外形 → 取 Payload[0..N-1]
        ↓ 扫描 TLV（代码字 + 数据）
emit rawReceived / signalQualityReceived / … / eegPowerReceived
```

- **不在此类** 做双 `AA` 组帧与校验和（组帧类已完成）。  
- **RAW `0x80`**：大端 16 位（高字节在前），与常见文档写法一致。  
- **`0x83` 八段功率**：与你在 `parserData` 中使用的 **下标 `i+2…i+25`** 一致（`0x83` 后第一字节跳过，随后 8×3 字节大端）。

## 二、`thinkgearpayloadparser.h`

```cpp
#ifndef THINKGEARPAYLOADPARSER_H
#define THINKGEARPAYLOADPARSER_H

#include <QByteArray>
#include <QObject>
#include <QString>
#include <QtGlobal>

/** 0x83 八个频段的 24 位无符号值（设备大端三字节拼成） */
struct ThinkGearEegPower {
    quint32 delta = 0;
    quint32 theta = 0;
    quint32 lowAlpha = 0;
    quint32 highAlpha = 0;
    quint32 lowBeta = 0;
    quint32 highBeta = 0;
    quint32 lowGamma = 0;
    quint32 midGamma = 0;
};

/**
 * 输入：一整帧（含 AA AA N payload checksum）。
 * 仅解析 N 字节 Payload 内的 TLV；越界则停止并发出 parseWarning。
 */
class ThinkGearPayloadParser : public QObject
{
    Q_OBJECT

public:
    explicit ThinkGearPayloadParser(QObject *parent = nullptr);

public slots:
    /** 连接 ThinkGearFrameAssembler::frameReady 到此槽。 */
    void parseFrame(const QByteArray &frame);

signals:
    /** 0x80：16 位原始波形（大端，按 qint16 解释） */
    void rawReceived(qint16 value);

    void signalQualityReceived(quint8 value);   // 0x02：Signal / 信号（思知大块；与 Neurosky Poor Signal 同源代码字）
    void attentionReceived(quint8 value);       // 0x04，0–100
    void meditationReceived(quint8 value);      // 0x05，0–100
    void blinkReceived(quint8 value);           // 0x16，1–255

    void eegPowerReceived(const ThinkGearEegPower &power); // 0x83

    void parseWarning(const QString &message);

private:
    /** 从整帧取出 N 字节 Payload；失败写 errorMsg。 */
    static bool extractPayload(const QByteArray &frame, QByteArray &payloadOut, QString *errorMsg);

    void parsePayload(const QByteArray &payload);
};

#endif // THINKGEARPAYLOADPARSER_H
```

## 三、`thinkgearpayloadparser.cpp`

```cpp
#include "thinkgearpayloadparser.h"

ThinkGearPayloadParser::ThinkGearPayloadParser(QObject *parent)
    : QObject(parent)
{
}

bool ThinkGearPayloadParser::extractPayload(const QByteArray &frame, QByteArray &payloadOut, QString *errorMsg)
{
    payloadOut.clear();

    if (frame.size() < 4) {
        if (errorMsg)
            *errorMsg = QStringLiteral("frame too short");
        return false;
    }

    if (static_cast<unsigned char>(frame[0]) != 0xAA || static_cast<unsigned char>(frame[1]) != 0xAA) {
        if (errorMsg)
            *errorMsg = QStringLiteral("bad sync");
        return false;
    }

    const int n = static_cast<unsigned char>(frame[2]);
    if (n == 0 || n > 170) {
        if (errorMsg)
            *errorMsg = QStringLiteral("invalid payload length");
        return false;
    }

    if (frame.size() != 4 + n) {
        if (errorMsg)
            *errorMsg = QStringLiteral("frame size mismatch");
        return false;
    }

    payloadOut = frame.sliced(3, n);
    return true;
}

void ThinkGearPayloadParser::parseFrame(const QByteArray &frame)
{
    QByteArray payload;
    QString err;
    if (!extractPayload(frame, payload, &err)) {
        emit parseWarning(err);
        return;
    }
    parsePayload(payload);
}

static quint32 readU24Be(const char *p)
{
    const auto b0 = static_cast<quint32>(static_cast<quint8>(p[0]));
    const auto b1 = static_cast<quint32>(static_cast<quint8>(p[1]));
    const auto b2 = static_cast<quint32>(static_cast<quint8>(p[2]));
    return (b0 << 16) | (b1 << 8) | b2;
}

void ThinkGearPayloadParser::parsePayload(const QByteArray &payload)
{
    int i = 0;
    const int n = payload.size();

    while (i < n) {
        const auto code = static_cast<unsigned char>(payload[i]);

        switch (code) {
        case 0x02: // 信号质量
            if (i + 1 >= n) {
                emit parseWarning(QStringLiteral("truncated 0x02"));
                return;
            }
            emit signalQualityReceived(static_cast<quint8>(payload[i + 1]));
            i += 2;
            break;

        case 0x04:
            if (i + 1 >= n) {
                emit parseWarning(QStringLiteral("truncated 0x04"));
                return;
            }
            emit attentionReceived(static_cast<quint8>(payload[i + 1]));
            i += 2;
            break;

        case 0x05:
            if (i + 1 >= n) {
                emit parseWarning(QStringLiteral("truncated 0x05"));
                return;
            }
            emit meditationReceived(static_cast<quint8>(payload[i + 1]));
            i += 2;
            break;

        case 0x16:
            if (i + 1 >= n) {
                emit parseWarning(QStringLiteral("truncated 0x16"));
                return;
            }
            emit blinkReceived(static_cast<quint8>(payload[i + 1]));
            i += 2;
            break;

        case 0x80: // RAW：高字节在前；有符号 int16（教程：>32767 时减 65536）
            if (i + 2 >= n) {
                emit parseWarning(QStringLiteral("truncated 0x80"));
                return;
            }
            {
                const auto hi = static_cast<quint8>(payload[i + 1]);
                const auto lo = static_cast<quint8>(payload[i + 2]);
                qint32 rawdata = (static_cast<qint32>(hi) << 8) | lo;
                if (rawdata > 32767)
                    rawdata -= 65536;
                emit rawReceived(static_cast<qint16>(rawdata));
            }
            i += 3;
            break;

        case 0x83: // 8 段 × 3 字节大端；与常见解析下标 i+2…i+25 一致
            if (i + 25 >= n) {
                emit parseWarning(QStringLiteral("truncated 0x83"));
                return;
            }
            {
                ThinkGearEegPower p;
                const char *base = payload.constData() + i;
                p.delta = readU24Be(base + 2);
                p.theta = readU24Be(base + 5);
                p.lowAlpha = readU24Be(base + 8);
                p.highAlpha = readU24Be(base + 11);
                p.lowBeta = readU24Be(base + 14);
                p.highBeta = readU24Be(base + 17);
                p.lowGamma = readU24Be(base + 20);
                p.midGamma = readU24Be(base + 23);
                emit eegPowerReceived(p);
            }
            i += 26;
            break;

        default:
            // 未知代码：无法得知长度，只前进 1 字节避免死循环；若设备有扩展表可在此补全
            ++i;
            break;
        }
    }
}
```

## 五、与组帧类、串口接线示例

```cpp
#include "device/serialport.h"
#include "thinkgear/thinkgearframeassembler.h"
#include "thinkgear/thinkgearpayloadparser.h"

auto *serial = new SerialPort(this);
auto *assembler = new ThinkGearFrameAssembler(this);
auto *parser = new ThinkGearPayloadParser(this);

connect(serial, &SerialPort::dataReceived, assembler, &ThinkGearFrameAssembler::onBytes);
connect(assembler, &ThinkGearFrameAssembler::frameReady, parser, &ThinkGearPayloadParser::parseFrame);

connect(parser, &ThinkGearPayloadParser::rawReceived, this, [](qint16 v) {
    // 写入 DataBuffer / 绘图（注意节流）
    Q_UNUSED(v);
});
connect(parser, &ThinkGearPayloadParser::eegPowerReceived, this, [](const ThinkGearEegPower &p) {
    Q_UNUSED(p);
});
connect(parser, &ThinkGearPayloadParser::parseWarning, this, [](const QString &w) {
    qWarning() << w;
});
```

## 六、说明与注意

| 项目 | 说明 |
|------|------|
| **双重校验** | 组帧类已验校验和；此处再检查 **帧长 == 4+N**，防止误连错误信号源。 |
| **`0x80` 的符号** | 与思知教程一致：`(hi<<8)|lo` 后 **`>32767` 则减 65536**；勿写成 `>32768` 漏掉 32768。 |
| **`default: ++i`** | 未知代码易错位；正式产品应对照官方 **EXCODE/扩展表** 补全长。 |
| **高频 RAW** | `rawReceived` 可能极快，UI 侧务必 **环形缓冲 + 定时刷新**，勿每点 `qDebug`。 |

## 七、与 `DataBuffer` 衔接（思路）

- **`rawReceived`**：`appendRawvalue(RawSample{…})`。  
- **`eegPowerReceived` + 注意力等**：合并进一次 `BigPackageSample` 再 `appendBigpackage`（或分信号各自更新成员后统一推一次，按你 UI 需求定）。
